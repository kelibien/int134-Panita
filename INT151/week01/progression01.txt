week1 พื้นฐานและเรื่องต่างๆที่อยากรู้หรือไม่เข้าใจ
8/8
- Number สามารถใช้แทนตัวเลขทั่วไปทั้ง int และ float
- String ใช้เก็บข้อความ จะใช้ '' หรือ "" ก็ได้
- ถ้าสร้างตัวแปรมาแล้วยังไม่ได้กำหนดค่าจะได้ Undefined เหมือนว่ามีสิ่งนั้นอยู่แต่ไม่สามารถหาค่าได้
- Null !=  Undefined -> Null ใช้แทนการกำหนดว่าตรงนั้นไม่มีค่าอะไรเลย
- ตัวแปรคนละตัวกันถึงแม้ว่าค่าจะเหมือนกันก็ไม่เท่ากันเพราะเป็นคนละตัวกัน

11/8
- const ใช้สำหรับค่าคงที่
- typeof ใช้สำหรับการตรวจสอบชนิดข้อมูล
- ถ้าใช้ + แล้ว number เจอ string จะเปลี่ยนเป็น string ทั้งหมด
- ถ้ารวม string และตัวแปรไว้ในบรรทัดเดียวกันสามารถใช้ `` แล้วแทรกด้วย ${ } ได้
- switch ตรวจค่าแล้วจับคู่กับ case คล้ายๆกับ === แล้วต้องใช้ break เพื่อหยุด

13/8
- For Loop -โครงสร้าง for (เริ่มต้น; เงื่อนไขต่อรอบ; อัพเดต)
           - ใช้วนตรวจสอบค่าที่ละครั้งว่าตรงกับเงือนไขที่เรากำหนดมั้ยถ้าไม่ตรงในจะละครั้งก็จะมีการอัพเดทเปลี่ยนแปลงค่าไปเรื่อยๆ
- While Loop - จะตรวจสอบเงือนไขก่อนแล้วค่อยทำตามในลูป
                        - ถ้าไม่ตรงเงื่อนไขตั้งแต่แรกจะไม่รันเลย
- Do–While - จะรันทำตามคำสั่งก่อนค่อยตรวจสองเงื่อนไข
